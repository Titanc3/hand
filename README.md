universal esp32/project controller

Preprogram a esp32 to intake 4 0-1 inputs --> "a:b:c:d"

(0 represents no flexion, 1 represents max flexion)
(you may need to tune the flex values depending on your flex sensors and hand movement)

add the mac address to the controller's device list

done! esp32 controller is ready to send data to your new project! :D
